스크롤 애니메이션

html에서 애니메이션을 넣을 요소를 class지정해줍니다.

css에서 .line은 처음 페이지에 접속했을 때의 line요소들의 상태를 말한다. opacity:0; 요소가 보이지 않도록한다. transform:translateY(20px):요소를 아래로 20px내려놓는다. transition :all 0.5s ease : 상태가 바뀔 때 0.5초동안 부드럽게 애니메이션 적용한다. .line.show 자바를 통해 .show클래스가 추가 되면, opacity:1 해당 요소가 보이게되고 transform: translateY(0) 원래 위치로 올라감

.line요소를 선택하여 lines변수에 저장한 뒤 스크롤이 발생할 때 실행할 함수에서 line을 한줄씩 순회시킨 후 요소가 화면 안에 들어왔는지 확인한다. 화면안에 요소가 있다면 .show요소를 붙여서 애니메이션이 실행된다.

섹션 기반 디자인 - 사용자 친화적 네비게이션

웹사이트는 하나의 HTML 파일 내에서 각 섹션을 명확하게 구분하여 설계되었습니다. 이를 통해 사용자는 페이지 내에서 상단 네비게이션 메뉴를 클릭하면 해당 섹션으로 부드럽게 이동할 수 있습니다. 이 방식은 페이지 로딩을 빠르게 하면서도, 사용자에게는 일관된 경험을 제공합니다.
예를 들어, '소개', '특징', '팀 구성', '문의'와 같은 주요 섹션들이 명확히 구분되어 있어 사용자는 원하는 정보로 빠르게 이동할 수 있습니다. 이 방식은 특히 긴 페이지에서 스크롤을 통해 각 섹션으로 자연스럽게 이동하는 데 유용하며, 페이지 내비게이션이 직관적으로 이루어집니다.
