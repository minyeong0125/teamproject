Section 3

@keyframes fadeInUp 이 애니메이션은 각 테이블 행이 점차적으로 위로 올라오며 나타나는 효과를 나타냅니다.
.ai-table tbody tr 이 부분은 .ai-table 클래스 내의 tbody 요소 안에 있는 모든 tr(테이블 행) 요소에 애니메이션을 적용합니다.
nth-child 선택자는 각 테이블 행에 대해 애니메이션이 순차적으로 지연되도록 설정합니다. 즉, 각 행의 애니메이션이 다른 시간에 시작됩니다.

스크롤 애니메이션

html에서 애니메이션을 넣을 요소를 class지정해줍니다.

css에서 .line은 처음 페이지에 접속했을 때의 line요소들의 상태를 말한다. opacity:0; 요소가 보이지 않도록한다. transform:translateY(20px):요소를 아래로 20px내려놓는다. transition :all 0.5s ease : 상태가 바뀔 때 0.5초동안 부드럽게 애니메이션 적용한다. .line.show 자바를 통해 .show클래스가 추가 되면, opacity:1 해당 요소가 보이게되고 transform: translateY(0) 원래 위치로 올라감

.line요소를 선택하여 lines변수에 저장한 뒤 스크롤이 발생할 때 실행할 함수에서 line을 한줄씩 순회시킨 후 요소가 화면 안에 들어왔는지 확인한다. 화면안에 요소가 있다면 .show요소를 붙여서 애니메이션이 실행된다.

Section 5

< div class="team-row" >< /div > 안에는 첫 번째 줄에 사진 3개만 넣는 class를 지정했다. 팀원 4부터 < div class="team-row second-row" >< /div > 안에는 두 번째 줄에 사진 2개만 넣는 class를 지정했다.

< div class="image-container" >는 팀원 사진과 오버레이(hover 시 나타나는 정보)를 감싸는 컨테이너이다.

< div class="overlay" > ... < /div >는 사진 위에 겹쳐지는 오버레이 영역입니다. 마우스를 올리면 이 부분이 파란색 배경과 함께 나타난다. < div class="overlay-box" >< /div >에는 오버레이 내부에 팀원 정보(이름, 역할)와 버튼들이 들어있다.

css의 overlay 부분은 기본적으로 opacity: 0으로 안 보이다가, transition: opacity 0.3s로 0.3초 동안 서서히 나타나게 한다. 그리고 border-radius: 150px로 원형 모양으로 만들었다. 그리고 hover를 이용하여 마우스를 버튼 위에 올리면 버튼 색이 검정색으로 변하고, 글자색이 하얀색으로 바뀐다.
